# Home depot system

### Dependencies
* make
* gcc, g++
* OpenSSL

### Development Environment
* NodeJS
* **docker**

### Install package
```
    npm install
```

### Run server
#### production
```
    npm run start
```
#### development
```
    npm run wrap
```
#### use pm2
```
    npm run cluster
```

### Find problems or synatx
```
    npm run lint
```

### Test
```
    npm run test
```
